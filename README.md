## Related Scripts

**1. Enter the project folder**
```
> Open a new terminal window and enter the folder
$ cd <project folder>
```
**2. Install packages via npm**
```
$ pnpm install
```
**3. For development**
```
$ pnpm dev
```
**4. Fix the coding style automatically**
```
$ pnpm lint
```
**5. Build Project**
```
$ pnpm build
/* output dir is dist*/
```
**6. Deploy on the server**
```
$ pnpm run deploy
```
